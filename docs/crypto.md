# Crypto (密码学)

**方向解释：**
密码学方向考察对各种加密算法、编码方式和数学原理的理解。题目通常给出一个加密后的密文或加密脚本，要求选手分析算法逻辑或利用数学弱点还原明文（Flag）。

**需要的学习内容：**
*   **基础知识**：基本数学（模运算、数论、群论、有限域）、编码格式（Base64, Hex, ASCII）。
*   **古典密码**：凯撒密码、维吉尼亚密码、栅栏密码、替换密码等。
*   **现代密码**：
    *   对称加密：DES, AES (ECB/CBC/CTR 模式), RC4。
    *   非对称加密：RSA (各种攻击场景), Diffie-Hellman, ECC (椭圆曲线)。
    *   哈希函数：MD5, SHA系列 (长度扩展攻击)。
*   **工具**：Python (主要工具), SageMath (处理复杂数学问题), yafu (大数分解), OpenSSL。
