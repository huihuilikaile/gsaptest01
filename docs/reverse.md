# Reverse (逆向工程)

**方向解释：**
逆向工程涉及对编译后的二进制程序进行分析，理解其内部逻辑和算法，从而推导出注册机、绕过验证逻辑或直接找到 Flag。通常不提供源代码，只提供可执行文件（Windows .exe, Linux ELF, Android .apk 等）。

**需要的学习内容：**
*   **基础知识**：计算机组成原理、操作系统原理、汇编语言 (x86/x64, ARM, MIPS)。
*   **编程语言**：C/C++、Python (用于写脚本)。
*   **静态分析**：能够阅读汇编代码，识别控制流结构（if, while, for）。
*   **动态调试**：使用调试器跟踪程序执行流程，查看寄存器和内存状态。
*   **工具**：
    *   静态分析：IDA Pro (核心工具), Ghidra, Hopper。
    *   动态调试：OllyDbg/x64dbg (Windows), GDB (Linux)。
    *   脱壳工具。
